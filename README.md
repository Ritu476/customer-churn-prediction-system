
Customer Churn Prediction System

This project builds an end-to-end churn prediction system that identifies customers likely to leave and provides actionable insights to improve retention strategies.


⚙️ Tech Stack

- Python (Pandas, NumPy)
- Machine Learning (Scikit-learn, XGBoost)
- Data Visualization (Tableau)



🔄 Project Pipeline

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (XGBoost)
- Model Evaluation
- Risk Segmentation


Dashboard Visualization

📊 Key Features

- Predicts customer churn using machine learning
- Generates churn probability scores for each customer
- Segments customers into High / Medium / Low Risk
- Identifies major churn drivers
- Interactive dashboard for business decision-making


📈 Dashboard Insights
- Customers with month-to-month contracts show highest churn
- Low tenure customers are more likely to leave
- Higher monthly charges correlate with increased churn risk


💡 Business Recommendations
- Offer discounts or incentives for high-charge customers
- Improve onboarding experience for new users
- Target high-risk customers with retention campaigns
